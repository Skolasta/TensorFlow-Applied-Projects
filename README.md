## 🔍 Detailed Project Breakdown

### [1. LangChain RAG Assistant (GenAI)](./langchain-rag-assistant/)
A modern, **fully local and private** document-based Question & Answering pipeline built with Retrieval-Augmented Generation (RAG). The project utilizes **LangChain** to connect an open-source `Llama-3` model (via Ollama) with a `ChromaDB` vector database, allowing users to upload technical PDFs and chat with them in real-time through a **Streamlit** user interface.

### [2. Pneumonia Detection CNN (Transfer Learning)](./pneumonia_Detection/)
An end-to-end computer vision pipeline that classifies chest X-ray images into Normal vs. Pneumonia classes. It demonstrates the capacity to combat class imbalance in medical datasets, augment image data, and implement **Transfer Learning** using a fine-tuned `ResNet50V2` architecture with optimized learning rates and callbacks.

### [3. Hybrid Intent Chatbot (NLP)](./hybrid-intent-chatbot/)
A robust intent classification neural network designed for automated customer service operations. By leveraging tokenization and customized deep neural architectures (saved natively as Keras models), the chatbot maps unstructured user queries to correct operational intents with high accuracy.

### [4. Industrial Casting Quality Control (Computer Vision)](./industrial-casting-quality-control/)
An automated visual inspection system for the manufacturing industry. The model detects microscopic defects in metal casting images using Deep Learning. It also comes packaged with a fast, lightweight **Streamlit** web application for immediate model demonstrations.

### [5. Customer Segmentation (Unsupervised Learning)](./customer-segmentation-ml/)
A foundational Data Science project focusing on tabular data wrangling and unsupervised machine learning. It covers the full analytical pipeline—from exploratory data analysis (EDA) and data cleansing with `Pandas`/`Seaborn`, to performing feature engineering and clustering techniques using `Scikit-Learn` to group customers by behavioral metrics.
