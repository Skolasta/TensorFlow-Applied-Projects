# Customer Segmentation using Machine Learning

This repository focuses on Customer Segmentation through Data Science and Machine Learning workflows, primarily utilizing TensorFlow and classical clustering strategies.

## 🚀 Features
- **Exploratory Data Analysis (EDA)**: In-depth data analysis to understand customer behavior patterns.
- **Data Preprocessing & Feature Engineering**: Cleaning and preparing complex datasets for machine learning models.
- **Machine Learning**: Utilization of TensorFlow and potential ML clustering algorithms to robustly group customers into distinct segments.

## 🛠️ Tech Stack
- **Machine Learning**: TensorFlow, Scikit-Learn
- **Data Analysis**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn
- **Development Environment**: Jupyter Notebook
